# Settings Hamburger Menu

The settings hamburger is a dedicated navigation control in the app chrome.

## Purpose

It keeps secondary navigation out of the main link row and gives the admin page a clear, intentional entry point.

## Current action

- opens a compact menu
- exposes the Admin local users screen

## Implementation

- component: `src/components/SettingsHamburgerMenu.vue`
- navbar wiring: `src/components/NavBar.vue`

## Usage

Click the settings button in the header, then choose Admin.
